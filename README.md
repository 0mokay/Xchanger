# Xchanger- It is a mozilla extension that replace text from a particular webpage (the webpage is a third party website), the webpage displays numbers arranged on html table

Summary of Xchanger 
Xchanger perform math equation on numeric values that is on the webpage when it is loading.
The Aim is to make sure the extension perform the task everytime the page is loaded and it must complete the task before the page is displayed 

the website vistore won't be seeing the actual figure but the result of the math equation
